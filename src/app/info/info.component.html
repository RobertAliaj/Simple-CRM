Welcome to the CRM WebApp

A simple and intuitive tool developed as a demonstration of coding skills. This application is a Customer Relationship Management (CRM) tool that operates with CRUD (Create, Read, Update, Delete) functionalities and is based on Angular. Firebase serves as the backend, while the user interface is designed using Angular Material, which is based on the principles of Material Design.


The main functions of the WebApp are as follows:

  • Dashboard: You can retrieve current information on the BTC price and the BTC market cap. These data are fetched through the integration of the CoinGeckoAPI.
  • User: Allows the addition of new users. Once created, you can click on the username to go to the User Details.
  • User Details: Here, you can edit user information and "send" BTC amounts.
  • Transactions: An area where you can view all conducted transactions.
  • Real-Time Crypto News: Thanks to the integration of the Gnews API, you can read daily real-time news from the world of cryptocurrencies here.

Note:

This tool mainly serves for demonstration and illustration of my skills; no real transactions or actions are possible. Enjoy exploring and testing!
